# Bachelorarbeit Mathematik
Ein vollständiger Beweis des (schwachen) Einbettungssatzes von Whitney, der garantiert, dass sich jede differenzierbare $n$-dimensionale Mannigfaltigkeit in den $$\mathbb{R}^{2n+1}$$ einbetten lässt. 

Enthält Beweise der Parakompaktheit differenzierbarer (zweitabzählbarer) Mannigfaltigkeiten, der Existenz abzählbarer und differenzierbarer Teilungen der eins und der grundlegenden Immersions- und Einbettungssätzen.
